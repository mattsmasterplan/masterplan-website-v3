MattsMasterPlan.com is an Angular 9 web application with each webpage being a TypeScript component. The styling is done using Angular Material Design with Bootstrap used to maintain mobile compatibility. Applications on this site use PHP and MySQL on the backend to interact with a database. All source code for this site is available on my GitHub.
